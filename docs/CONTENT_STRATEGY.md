## Content Principles

### 1. Clarity Over Cleverness
Say what we mean. No jargon, no buzzwords, no "reimagining" or "ecosystem" language.

### 2. Honest About Status
Distinguish between:
- **Available now:** AbiliLife Mobility pilot in Nairobi
- **Coming later:** Care, Work, Access, Learn modules

### 3. Mission-Led, Not Feature-Led
Lead with purpose, not feature lists. Why we exist matters more than what buttons we have.

### 4. Grounded in Place
We are in Nairobi, Kenya. Not "across Africa" (yet). Specificity builds credibility.

### 5. Human Voice
Write like a person explaining to a friend, not a company writing marketing copy.

## Forbidden Language

Do not use these words/phrases anywhere on the site:

| Forbidden | Why |
|-----------|-----|
| "Super app" | Overpromises current state |
| "Ecosystem" | Vague corporate language |
| "Reimagining" | Buzzword |
| "Revolutionary" | Hyperbole |
| "Across Africa" | Not accurate (Nairobi only) |
| "24/7 support" | Not verified |
| "GPS tracking" | Not implemented |
| "Trained drivers" | We don't own drivers |
| "One-tap booking" | Not the current UX |
| "Instant" | Not accurate |
| "Comprehensive" | Overstates completeness |
| "Empowering" | Overused, vague |
| "Download on Play Store" | Not on Play Store (APK only) |
| "Available on App Store" | No iOS version |

---

## Allowed Language

These are safe, honest alternatives:

| Use This | Instead Of |
|----------|------------|
| "Accessible rides" | "Inclusive mobility solutions" |
| "Wheelchair users" | "Persons with disabilities" (when specific) |
| "Partner vehicles" | "Our fleet" |
| "We're testing" | "We've built" |
| "Join our pilot" | "Download the app" (when referring to Play Store) |
| "Download the pilot app" | "Get it on Google Play" |
| "Android APK" | "Mobile app" (be specific about platform) |
| "Manual install" | Avoid hiding the friction |
| "Early access" | "Full release" |
| "Chat with us" | "Contact our support team" |
| "Nairobi, Kenya" | "East Africa" / "Africa" |
| "Early-stage" | "Comprehensive" |
